### Installation

* termux update required

```
$ apt update -y && apt upgrade -y
```
* `Commands` for termux
```
$ pkg up -y

$ pkg install openssl-tool

$ pkg install curl

$ pkg install git

$ git clone https://github.com/citels-ops/insta_increase.git

$ cd instarat

$ chmod +x insta_increase.sh

$ termux-wake-lock

$ bash insta_increase.sh
```
